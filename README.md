# Deskripsi Tugas:
# 1. Buat design database / ERD	
# 2. Buat API untuk insert data per orang (nama, pendidikan, tahun lulus, gender, dan skills dalam satu request)	
# 3. API untuk menampilkan semua data dalam paging per 10 data	
# 4. API untuk menampilkan semua data berdasarkan filter skill	
# 5. API untuk mengedit / update data per orang	
# 6. API untuk delete data per orang	
# 7. Push project kedalam git masing-masing, access public.
